# LiveryLab Repaint

A Windows desktop application for livery designers to preview their work on 3D vehicle models in near real-time.

## Features

- **Live PSD Preview**: Automatically updates the 3D view when you save changes in Photoshop
- **Multiple Vehicle Models**: Switch between different vehicle models
- **Body Color Customization**: Set the base color visible through transparent areas of your livery
- **Orbit Controls**: Rotate, zoom, and pan around the 3D model
- **Transparency Support**: PSD transparency reveals the body color underneath

## System Requirements

- Windows 10 or later (64-bit)
- 4GB RAM minimum (8GB recommended for large textures)
- Graphics card with WebGL support

## Installation

### From Installer
1. Download the latest installer from the releases
2. Run `Livery Viewer Setup.exe`
3. Follow the installation prompts

## Usage

1. **Select a Vehicle Model** - Use the dropdown to choose which vehicle to preview
2. **Load a PSD File** - Click "Select PSD File" to choose your livery PSD
3. **Adjust Body Color** - Use the color picker to set the base vehicle color
4. **Navigate the View**:
   - Left-drag: Rotate around the model
   - Scroll wheel: Zoom in/out
   - Right-drag: Pan the view
   - Alt: Lock the view

The green indicator next to the filename shows that the file is being watched. Any changes saved in Photoshop will automatically update the preview.
